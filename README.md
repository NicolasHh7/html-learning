## HTML
### Conceito
<ul>
  <li>HTML é uma linguagem de marcação responsável pela estrutura de um site.</li>

  <li>
    <strong>W3C</strong> – Organização que buscou padronizar o HTML, pois no passado havia dificuldade para atualizar e manter compatibilidade entre navegadores.
  </li>

  <li>
    <strong>2007–2008</strong> – Surgimento do HTML5, focado em Web Semântica, tornando o código mais significativo e organizado.
  </li>
</ul>

# Estruturas Web

## 1. Estrutura básica do HTML

### **`<!DOCTYPE html>`**

* Informa ao navegador o tipo de documento (HTML).

### **`<html lang="pt-BR">...</html>`**

* Define o idioma do conteúdo da página.

### **`<head>...</head>`**

* Contém configurações e informações da página.
* Elementos comuns:

  * **`<meta charset="UTF-8">`** — Define quais caracteres podem ser usados.
  * **`<meta http-equiv="X-UA-Compatible">`** — Compatibilidade com navegadores.
  * **`<title>...</title>`** — Título do site (aparece na aba).

### **`<body>...</body>`**

* Contém todo o conteúdo visível da página.

## 3. Ferramentas do Console do Desenvolvedor

Acessado clicando em **Inspecionar** no navegador.

* **Elements** — Estrutura HTML e CSS.
* **Console** — Mensagens, logs e testes com JavaScript.
* **Sources** — Arquivos utilizados pela aplicação.
* **Network** — Monitoramento de requisições.
* **Performance** — Métricas de desempenho.
* **Memory** — Análise de uso de memória.
* **Application** — Armazenamento local, cookies e mais.
* **Security** — Segurança da página.
* **Lighthouse** — Auditoria de qualidade da página.

### Observação

* Permite testar **responsividade** em diferentes tamanhos de tela.
